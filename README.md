CISINB est un projet open‑source conçu pour faciliter l’audit de conformité et la sécurité des systèmes en s’appuyant sur les recommandations du Center for Internet Security (CIS).
Il fournit une structure normalisée pour définir, exécuter et documenter des contrôles de sécurité, avec une logique adaptable à différents environnements (PostgreSQL 15, Maria DB 10.6, SQL server 2022,Oracle DB 19c)

Ce dépôt sera mis à jour en fonction de mes audits :)

## Run Locally

**Prerequisites:**  Node.js

1. Copy git
 `git clone https://github.com/mchenock/CISINB `
 `cd CISINB `

2. Install dependencies:
   `npm install`
3. Run the app:
   `npm run dev`
4. Open Browser
 ` http://localhost:3000 `
